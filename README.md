# Tucil3_13524057

## About this template
Ice Sliding Puzzle Solver is a program to find an optimal solution of an ice sliding puzzle.


## Requirements

### C++ Version
- **Version:** C++17 or higher

### CMake Version
- **Version:** 3.28.3 or higher

## How to Run

### Linux
```bash
cmake --build build
./build/IceSliderSolver
```
